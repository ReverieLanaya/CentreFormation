## Centre Formation

This is an app that sells courses online with differents topics. Fon now, you can only consult wich courses are saved in the database based on workplace, keyword or all the courses.


## How to launch the app
You need to be located in the eval_java repository and go to : eval_java\Centre Formation\src\app then launch the FormationApp.java file

## Functions as of today : 
 - checkAllFormations : takes no parameter, display all of the courses curently in the database
 - checkFormationWorkplace : takes one parameter, display courses where the workplace match the parameter (in this case Distanciel or Présentiel)
 - checkFormationKeyword : takes one parameter, display courses where the description contain a word or a chain of letters that match the parameter

## The options in the console

1. Afficher toutes les formations disponible -> print all courses
2. Afficher les formations en présentiel -> print in-person courses 
3. Afficher les formations en télétravail -> print remote courses 
4. Rechercher des formations par mot-clés  -> print courses where the description contains the keyword
5. Consulter son panier -> can not be used yet so it will just print a message
6. Créer un compte -> can not be used yet so it will just print a message
7. Quitter -> will stop the program
